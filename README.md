# README
## 🏠 House Price Prediction Project

### 📌 Project Overview
In this assignment, I work as a Data Analyst at a Real Estate Investment Trust.  
The goal is to support the company in entering the residential real estate market by building a model that can estimate house prices based on various features.

---

### 📊 Dataset Description
The dataset contains house sale prices for **King County (including Seattle, USA)**.  
It includes transactions from **May 2014 to May 2015**.

This dataset has been slightly modified for learning purposes.

Key features include:
- Square footage
- Number of bedrooms
- Number of floors
- Other relevant housing attributes

---

### 🎯 Objectives

- Analyze housing data to understand price-driving factors  
- Perform data cleaning and preprocessing  
- Conduct Exploratory Data Analysis (EDA)  
- Build models to predict house prices  
- Complete all analytical tasks provided in the notebook  

---

### 🧪 Tools & Environment

- JupyterLab (Skills Network Labs – Cloud environment)  
- Python (Pandas, NumPy, Matplotlib, Seaborn)  
- Scikit-learn
